Divi SUPERCHARGER adds tons of great options to the Divi plugin.  Check out the Theme Customizer under Divi SUPERCHARGER Options. Choose new blog layouts, hover effects, link hover colors, slider navigation, additional menu options and more!

== Description ==

Divi SUPERCHARGER adds great features to your Divi installation:

* Hover link color
* Footer hover link color 
* Blog layouts 
* Image highlight hover effects 
* Slider navigation colors 
* Primary menu options like submenu highlighting
* Support for automatic resize if the menu spans two or more rows
* Apply the Divi styles to more form elements & buttons
* New features added regularly!

== Installation ==

= Minimum Requirements =

* PHP version 5.3 or greater (PHP 5.6 or greater is recommended)
* WordPress 4.6+
* Divi v3.0+

= Automatic installation =

If you don't already have the plugin installed, simply download the plugin from your [account](https://dustysun.com/my-account/). Next navigate to the Plugins menu in WordPress and click Add New. Upload your new plugin and then activate it!

= Updating =

Automatic updates should work like a charm; as always though, ensure you backup your site just in case.
